# 個人品牌關聯圖

Nicolas 的自我探索儀表板 — 把「輸入 → 內化 → 輸出 → 目標」做成一張會長大的動態關聯圖。

- 程式碼在此 public repo（GitHub Pages 部署）
- 節點/連線/反思筆記等資料存在 **private** repo `nicolas-brand-graph-data`，需 fine-grained PAT 才能讀寫
- 首次開啟網站 → 右上「鑰匙」貼上 PAT 即載入

技術：單檔 HTML + [force-graph](https://github.com/vasturiano/force-graph)，資料走 GitHub Contents API。
